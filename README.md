# UBB Plan

Aplikacja mobilna z planem zajęć dla **Uniwersytetu Bielsko-Bialskiego**.



## Pobierz

[![Download APK](https://img.shields.io/badge/Download-APK-green?style=for-the-badge&logo=android)](https://github.com/matikgal/ubb-schedule/releases/latest)

## Technologie

- **React 19** + **TypeScript**
- **Vite** - bundler
- **Capacitor** - natywna aplikacja Android
- **Supabase** - backend
- **Framer Motion** - animacje
- **Lucide React** - ikony
